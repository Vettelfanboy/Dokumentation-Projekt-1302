### 1.2 User Stories

| US-№ | Verbindlichkeit | Typ  | Beschreibung                       |

| 2    |  Muss           | Funktional | Als ein User möchte ich, dass ich informiert werde, ob meine Antwort richtig oder falsch ist. |
